# Scripts

This folder contains helper scripts to install dependencies and run the dashboard.

- `run_dashboard.ps1` — PowerShell script to install requirements (if present) and start the dashboard on Windows.
- `run_dashboard.sh` — Bash script for Unix/WSL to install and run the dashboard.
- `install_requirements.ps1` / `install_requirements.sh` — Install dependencies from `requirements.txt`.

Usage examples are in the top-level `docs/usage.md`.
